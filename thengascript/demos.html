Ever wondered what it would be like to program for the web in Malayalam? That is what drove the creation of Thenga(Coconut)script. With it you can start programming for the web in Malayalam.

Underneath the hood, it translates your code to it's ECMAscript equivalent which is then interpreted and ran by the browser.

Usage

Include the code as:

      <script type="text/thengascript">
      /* Write your thengascript code here */
      </script>

or you can also include the file using the src attribute

      <script type="text/thengascript">
      /* Write your thengascript code here */
      </script>
